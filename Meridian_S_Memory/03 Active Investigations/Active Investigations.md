# Active Investigations

## Purpose

This note represents the present-tense layer of the memory system.

It is not a backlog.
It is a compressed view of what is currently alive in the research.

## Current High-Level Focus

### 1. Building Better Research Memory

The immediate meta-project is the creation of a memory environment that can support multiple agents without prematurely freezing identity or behavior.

### 2. Agentic Research Infrastructure

The research remains strongly engaged with:

- agent frameworks
- memory substrates
- routing logic
- local and sovereign execution stacks
- observability and evaluation layers
- spec-driven execution patterns

### 3. Tool Evaluation As Strategic Practice

Tools are not being studied casually.
They are being evaluated as possible parts of a long-term operating system.

### 4. Project Translation

The research still values the conversion of ideas into concrete systems and products.

### 5. Project Layer Expansion

The memory system now has a stronger influx of live project material:

- Doodle Scribe
- FocusFlow
- Keyword Sleuth
- Shady Clause Detector
- Prompt CLI / Aider / Qwen execution work

This means project memory is becoming a more important carrier of research direction.

## Current Productive Tension

The most important tension right now is:

How do we create an agent memory strong enough to guide useful work without turning an evolving research identity into rigid law too early?

A second major tension is now emerging:

How do memory, routing, observability, and execution contracts fit together into one coherent agentic system without becoming tool bloat?

## What Agents Should Infer

- the system is still early enough to remain flexible
- structure is welcome, rigidity is not
- novelty should come from context, not from random divergence
- the memory environment itself is now part of the research
- the execution stack is becoming more formal and more inspectable

## Immediate Priority

Strengthen the memory substrate so that future research and brainstorming agents can operate with continuity, context, and non-repetitive awareness.

Immediate next structural priority:
integrate the new project and agent-construction deltas without sacrificing compression.

# Project Alignment

## Current Project Gravity

The strongest project gravity is around systems that connect:

- research
- memory
- agents
- publishing
- creative tooling

## Alignment Rule

Good work in this ecosystem should usually strengthen at least one of these:

- research continuity
- agent capability
- memory structure
- strategic execution
- meaningful creative expression

## Misalignment Signal

If a direction is only technically interesting but does not connect back to the larger system, it should be treated cautiously.

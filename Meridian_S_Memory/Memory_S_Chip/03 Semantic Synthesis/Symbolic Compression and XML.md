# Symbolic Compression and XML

## Core Principle

Structure helps preserve meaning under compression.

## Two Useful Forms

### Symbolic Compression

Use shorthand, symbols, and compact operators when they genuinely preserve a rich concept.

### XML-Like Tagging

Use tags to force extraction discipline and reduce noise in long texts.

## Practical Reminder

Compression is useful only when it remains explainable and reusable.

If the shorthand becomes private fog, it has failed.

# Semantic Synthesis Core

## Core Claim

The work aims at knowledge synthesis, not just information retrieval.

## Main Techniques

### Roundtable Reasoning

Use debate or dialogue between thinkers or positions to expose deeper structure.

### Analogical Mapping

Look for structural isomorphisms across domains.
This is where real synthesis often begins.

### Red Teaming

Stress-test insights by searching for:

- missing links
- edge-case failures
- opposing evidence

### Structured Extraction

Force reading into stable fields such as:

- concept
- tension
- personal link
- philosophical seed

## Strategic Value

These methods reduce passive consumption and increase the density of thought produced from each source.

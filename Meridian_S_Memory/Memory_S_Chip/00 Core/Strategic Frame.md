# Strategic Frame

## Core Reality

The work is not ordinary note-taking or generic AI use.

It is the design of a research-and-action system around:

- agentic execution
- routing and observability
- semantic synthesis
- durable memory and context
- creative and philosophical depth

## Main Strategic Distinctions

### Planning is not execution

Heavy reasoning models should define the contract.
Bounded tools or lighter models should execute it.

### Memory is not archive

Archive stores.
Memory organizes.
Short memory primes.

### Observability is not optional

A system that cannot inspect itself becomes guesswork.

### Compression is not simplification

Good compression preserves the distinctions that change action.

## Short-Memory Use

If a new note does not improve immediate orientation or preserve a core distinction, it does not belong in this layer.

# Memory_S_Chip

## Purpose

This folder is a short-memory layer for high-value recurring context.

It exists to make the right ideas immediately available at the start of work.

## What It Preserves

- the strategic frame of the work
- the key agent-building patterns
- routing and observability logic
- synthesis methods that improve reasoning
- the minimum personal and project context needed for alignment

## What It Does Not Try To Be

- a full archive
- a complete memory system
- a product manual
- a biography

# Memory Aware Agents

## Core Idea

The most useful agents are not just tool users.
They are agents that can operate with continuity.

## What Continuity Requires

- a stable core context
- a readable project and identity layer
- explicit memory boundaries
- structured updates instead of raw accumulation

## Lifelong Growth Principle

An agent that grows with the user should:

- preserve preferences and constraints
- improve from prior successes and failures
- maintain structured memory instead of opaque drift

## Practical Rule

Memory should help the agent do better work in the next session.
If it only makes the system heavier, it is not useful memory.

# Core Research Compressions

## Investigation Operating System

Research should end in:

- a decision
- a reusable note
- or an operational next step

## Agentic Systems

Agents matter when the work contains loops, tools, memory, or repeated execution.
They do not matter merely because they sound advanced.

Spec-driven execution is becoming a preferred pattern:
heavy models define the contract, bounded models or tools execute, and verification closes the loop.

## Tool Intelligence

Tool study is valuable only when it leads to judgment:

- fit
- non-fit
- replacement value
- architectural consequence

Routing and observability now count as first-class architecture, not support utilities.

## Knowledge Architecture

A strong knowledge system compresses and connects meaning rather than merely storing text.

## Strategic Filters

The system values negative knowledge.
Knowing what to avoid is part of knowing what to build.

## Project Memory

Projects are now one of the strongest memory carriers in the system.
They show what the research is willing to embody, not just what it is willing to discuss.

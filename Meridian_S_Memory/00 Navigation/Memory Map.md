# Memory Map

## Layer Logic

This memory environment has eight layers.

### `00 Navigation`

Entry points and reading order.

### `01 Research Identity`

A provisional sketch of your recurring concerns, ambitions, and research temperament.

### `02 Research Terrain`

The durable subject landscape.
This is where the memory environment inherits the best long-term structure from `Code Archive 2`.

### `03 Active Investigations`

The present tense of the system.
This is where the memory environment inherits the strongest active-context instinct from `LUX`.

### `04 Project Memory`

The project layer.
This preserves the memory of projects as embodiments of research priorities.

### `05 Idea Frontier`

The novelty layer.
This tracks unresolved tensions, promising adjacencies, and what should not be repeated.

### `06 Reference Compression`

Fast-reading packets for agents.
These are compact but high-signal.

### `07 Memory Protocols`

The logic of how this memory works, how it should be updated, and how stable files differ from living files.

## Mental Model

This system is neither a vault index nor a personality bible.
It is a staged memory substrate:

- identity gives direction
- terrain gives structure
- active investigations give urgency
- project memory gives embodiment
- idea frontier gives motion

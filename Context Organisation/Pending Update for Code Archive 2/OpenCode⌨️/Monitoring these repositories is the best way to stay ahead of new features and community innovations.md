While the official OpenCode repository is the clear flagship, a vibrant ecosystem of complementary projects has grown around it. Monitoring these repositories is the best way to stay ahead of new features and community innovations.

### The Core Project
This is the primary open-source AI coding agent that all other projects are built upon.
* **Repository:** [anomalyco/opencode](https://github.com/anomalyco/opencode)
* **Stars:** ~140,000+
* **Maintainer:** Anomaly
* **Key Features:** A powerful terminal-based AI assistant that reads project files, writes and edits code, and runs commands. Its rapid growth has made it the leading open-source challenger to tools like Claude Code.

### 🤖 Full Desktop & Computer Use Agents

These applications aim to give an AI agent broad control over your entire computer environment, including clicking, typing, and using any software.

- **RealOS**: A self-hosted AI desktop agent that automates tasks using natural language within a secure, containerized Linux desktop environment.
    
- **Open Computer Use**: An open-source platform that gives AI agents the ability to control a computer to browse the web, run commands, and perform multi-step workflows in isolated containers.
    
- **Accomplish**: A free, open-source AI desktop agent for Windows, macOS, and Linux. It can use a built-in local AI for privacy, requiring no API key, and can also connect to external models.
    
- **Aulinx**: An AI agent specifically designed to control a Linux desktop via natural language, with a focus on UI interaction using the AT-SPI accessibility framework.
    

### 💻 Terminal-First AI Coding Assistants

These are command-line interface (CLI) tools that act like a senior developer pair-programming with you directly in your terminal.

- **Claude Code**: Anthropic's own agentic coding tool that lives in your terminal, understands your codebase, and helps you code faster by executing routine tasks.
    
- **Aider**: A popular AI coding assistant that runs as a terminal CLI. It can edit files, run shell commands, and integrates with Git, all through a chat-based REPL interface.
    
- **Agentic Terminal**: An open-source, folder-scoped AI agent that acts as a senior DevOps engineer in your terminal, capable of reading files, editing code, and running shell commands with your approval.
    
- **Poseidev**: An autonomous tool where the AI can edit, create, and rename files, as well as run tests, builds, and development servers from the terminal.
    
- **Codestral**: An AI-powered terminal coding assistant that integrates with Mistral AI models to provide code assistance and autonomously execute tasks using agentic capabilities.
    
- **ColdCode**: An AI assistant that lives in your terminal and can understand your codebase, edit files, run commands, and handle entire workflows for you.
    

### 🔧 Alternative MCP Servers & Developer Tools

These are often smaller, more specialized tools that integrate with the Model Context Protocol (MCP) to give AI assistants specific capabilities.

- **Terminal Controller**: An MCP server that provides a secure interface for Large Language Models (LLMs) to execute terminal commands, navigate directories, and perform file system operations.
    
- **wcgw**: A Python package that provides an MCP server with tightly integrated shell and code editing tools, focusing on safety by requiring the AI to read a file before editing it.
    
- **MCP Manager**: A tool that allows you to execute terminal commands in different contexts (local, SSH, and GitHub) through a single MCP server.
    
- **nlsh-ai**: A tool-enabled AI for the terminal that can perform file operations and execute shell commands, with mandatory confirmations for safety.
    
- **Ledit**: An AI-powered code editing and assistance tool with a Web UI that includes a chat interface, code editor, terminal, file browser, and Git UI.
